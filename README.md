# Knight Tour Trainer — Support

This repository contains support information for the **Knight Tour Trainer** mobile app (iOS).

## Contact
- Email: **(will be added soon — Gmail)**
- GitHub issues: Please open an issue in this repository.

## What to include in a support request
- App version (e.g., 1.2.0)
- Device model (e.g., iPhone 16 Plus)
- iOS version
- Steps to reproduce the problem
- Screenshots or screen recording (if possible)

## Diagnostics
The app uses **Firebase Crashlytics** (crash reports) and **Firebase Analytics** (basic usage analytics) to help improve stability and understand how features are used.

## Advertising
The current version does **not** show ads. Advertising may be added in future versions; this page and the privacy policy will be updated before such features are released.
